# MovieNotes-Front-end

Me aprofundando no Front-end com React.js! Utilizando o Vite.js para criar um ambiente de Desenvolvimento simples e super rápido. Aqui é uma aplicação SPA(single page application), com o React está sendo possível criar componentes, permitindo dividir a UI em partes independentes, reutilizáveis. E com as propriedades, estou mudando o valor de cada componente, permitindo ser diferente em cada pagina. Styled-components está sendo usado para escrever o CSS dentro do javascript, para aplicar um ótimo desempenho e uma manutenção mais tranquila. 👨‍💻📚
